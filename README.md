# Personal-Assistant-Python
Personal speech assistant to automate few simple tasks by voice input built using python.
## List of tasks it can do
1. Telling the time and date.
2. Searching any term on Google.
3. Finding any location on Google Maps.
4. Searching for any video on YouTube.
5. Fetching any Wikipedia article.
6. Opening any application installed in Windows.
7. Opening few famous websites on web browser.
   * Facebook
   * Youtube
   * Google
   * Gateoverflow
   * Amazon
   * Gmail
   * Twitter
   * Unacademy
8. Shopping items from 3 different e-commerce websites
   * Amazon
   * Flipkart
   * Myntra
